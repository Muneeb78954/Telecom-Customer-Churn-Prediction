# Telecom Customer Churn Prediction

Developed a machine learning solution to predict telecom customer churn, achieving **95.98% accuracy** with an **F1 Score of 0.8057**.

## Project Highlights

* Built and evaluated an ensemble benchmarking pipeline using:

  * **CatBoost**
  * **XGBoost**
  * **LightGBM**
* Performed **5-fold cross-validation** for each model to ensure robust and reliable performance.
* Implemented automated model comparison and selection to identify the best-performing algorithm based on evaluation metrics.
* Focused on maximizing predictive performance while maintaining strong classification quality for churn detection.

## Performance

* **Accuracy:** 95.98%
* **F1 Score:** 0.8057
